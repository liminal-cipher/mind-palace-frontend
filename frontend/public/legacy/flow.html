5체험 (방 안)장소법으로 기억
🚶 1인칭 워크스루
memory-walk.html
방 GLB를 WASD로 걷고, 가구·지점(핫스팟)에 학습 개념이 얹힘. 🧠 개념지도(2D 대시보드)로 배치 편집. 좌상단 처음/지도/뒤로.
🏠 처음=home🗺 지도=vworld_map🧠 개념지도
D데이터 · 파이프라인화면 아래에서 자동으로 도는 것
📄 PDF→외부 GraphRAG→palace.json→localStorage mp_palace→ compose·studio·map·walk 모두 같은 자료 사용
🏠 새 방 GLB→memory-walk 런타임→ 가구 메쉬 좌표 자동추출 → 부족하면 지오메트리 폴백 → 노드 수만큼 핫스팟 합성 → localStorage 편집 저장
🖼 무명/단일메쉬 GLB→스캐너 personal-room-scanner-3d→hotspots.json→ 정밀 좌표(자동 실행 예정)
📍 랜드마크→ 미리 준비한 84개 도시 데이터(cities/*.json)에서 도시별 배정
🎨 방 배정→ 스튜디오 선택은 localStorage studio_designs → 지도·입장이 그 방을 사용
전제 — 다른 PDF가 들어올 수 있고, 새 방 GLB도 받을 수 있다는 가정 위에서: 활성 자료(mp_palace)는 어느 화면으로 가도 유지되고, 어떤 GLB든 입장 시 좌표가 런타임에 계산돼 핫스팟이 붙는다. Claude CLI 개입 없이 동작.
